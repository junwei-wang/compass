# Tmux Cheetsheet

* TOC
{:toc}

start new session:

```shell
tmux new -s ${session-name}
```

attach to an existing session:

```shell
tmux attach -t ${session-name} 
# OR
tmux a -t ${session-name}
```

## configuration

Edit `~/.tmux.conf`

```shell
# bind new leader key
unbind-key C-b
set -g prefix 'C-q'
bind-key 'C-q' send-prefix
```

## window (tabs)

```shell
c   create window
w   list windows
n   next window
p   previous window
```
